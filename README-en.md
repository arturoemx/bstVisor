# bstVisor

An implementation of a visual exploration tool for Binary Search Trees.

Currently two versions for two different libraries graphics libraries have been tested:

 * **simple2D**: '*Simple 2D is a small, open-source graphics engine providing essential 2D drawing, media, and input capabilities. It's written in C and works across many platforms, creating native windows and interacting with hardware using SDL while rendering content with OpenGL'*.
   *  [simple2D](https://github.com/simple2d/simple2d)
	 
 * **raylib**: '*raylib is a simple and easy-to-use library to enjoy videogames programming.*'
   * [raylib home page](https://github.com/raysan5/raylib)
   * [raylib github page](https://www.raylib.com/)
